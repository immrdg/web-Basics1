Python As A Calculator

Python is perfectly suited to do basic calculations. Apart from addition, subtraction, multiplication and division, there is also support for more advanced operations such as:

  • Exponentiation: **. This operator raises the number to its left to the power of the number to its right.
    For example 4**2 will give 16.
  • Modulo: %. This operator returns the remainder of the division of the number to the left by the number on its right.
    For example  18 % 7 equals 4.
      print(1+2)
#output>> 3

print(3**2)
#output>> 9

print(10%7)
#output>> 3

Data Types

In programming, data type is an important concept.

Variables can store data of different types, and different types can do different things.

Python has the following data types built-in by default, in these categories:

Text Type: Str
Numeric Types:  int, float, complex
Sequence Types:  list, tuple, range
Mapping Type:  dict
Set Types:  set, frozenset
Boolean Type:  bool
Binary Types:  bytes, bytearray, memoryview

Basic Data Types

Integers

In Python, integers are zero, positive or negative whole numbers without a fractional part and having unlimited precision,
e.g. 0, 100, -10.Integers can be binary, octal, and hexadecimal values.

Float

In Python, floating point numbers (float) are positive and negative real numbers with a fractional part denoted by the decimal symbol . or the scientific notation E or e, e.g. 1234.56, 3.142, -1.55, 0.23.
